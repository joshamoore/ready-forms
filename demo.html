Ready Forms 0.1

Ready Forms is a complete CSS & JavaScript form system, which makes it easy to add beautiful forms with built in validation to your website. Simply add the ready-forms class name to your inputs and the rest is done for you!

Current Features

  • Styling for text, email and textarea input types
  • Validation for text and email input types

Supported Input Types

  • text
  • email
  • textarea

Usage

Just copy the ready-forms folder into your website root, and include the following lines your head:

  • <link rel='stylesheet' type='text/css' href='ready-forms/css/ready-forms.css' />
  • <script type="text/javascript" src="ready-forms/js/ready-forms.js"><script>

Form styling in ready-forms/css/ready-forms.css

Form validation in ready-forms/js/ready-forms.js

Supported inputs have a corresponding class which follows the naming convetion ready-forms-[input type]. Simply add that to your form and styling and validation will be added. Refer to the list of supported input types for a full list of class names.

Default width of text inputs is 300px. You can change this in the css/ready-forms.css file.

Input Type Class Names

  • text - ready-forms-text
  • email - ready-forms-email

If you want a required field simply add -required at the end of a class name.

Eg. ready-forms-text-required will add validation to a text field.
